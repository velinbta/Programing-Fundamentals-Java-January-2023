7. Max Sequence of Equal Elements
Write a program that finds the longest sequence of equal elements in an array of integers. If several longest sequences exist, print the leftmost one.

Examples:

Input 1:
2 1 1 2 3 3 2 2 2 1

Output 1:
2 2 2

Input 2:
1 1 1 2 3 1 3 3

Output 2:
1 1 1

Input 3:
4 4 4 4

Output 3:
4 4 4 4

Input 4:
0 1 1 5 2 2 6 3 3

Output 4:
1 1
