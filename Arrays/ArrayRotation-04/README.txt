4. Array Rotation
Write a program that receives an array and the number of rotations you have to perform (the first element goes at the end). Print the resulting array.

Input 1:
51 47 32 61 21
2

Output 1:
32 61 21 51 47

Input 2:
32 21 61 1
4

Output 2:
32 21 61 1

Input 3:
2 4 15 31
5

Output 3:
4 15 31 2
