2. Common Elements
Write a program that prints common elements in two arrays. You have to compare the elements of the second array to the elements of the first.

Input 1:
Hey hello 2 4
10 hey 4 hello

Output 1:
4 hello

Input 2:
S of t un i
of i 10 un

Output 2:
of i un

Input 3:
i love to code
code i love to

Output 3:
code i love to
