1. Ages
Write a program that determines whether a person is based on the given age: baby, child, teenager, adult, or elder. The bounders are:

0-2 - baby; 
3-13 - child; 
14-19 - teenager;
20-65 - adult;
>=66 - elder; 
All the values are inclusive.

Examples:

Input 1:
20

Output 1:
adult

Input 2:
1

Output 2:
baby

Input 3:
100

Output 3:
elder
