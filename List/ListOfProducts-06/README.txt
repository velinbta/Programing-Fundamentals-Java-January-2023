6. List of Products
Read a number n and n lines of products. Print a numbered list of all the products ordered by name.

Examples:

Input 1:
4
Potatoes
Tomatoes
Onions
Apples

Output 1:
1.Apples
2.Onions
3.Potatoes
4.Tomatoes

Input 2:
3
Orange
Grape
Strawberry

Output 2:
1.Grape
2.Orange
3.Strawberry
