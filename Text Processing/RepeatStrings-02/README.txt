2. Repeat Strings
Write a Program That Reads an Array of Strings. Each String is Repeated N Times, Where N is the length of the String. Print the Concatenated String.

Examples:

Input 1:
hi abc add

Output 1:
hihiabcabcabcaddaddadd

Input 2:
work

Output 2:
workworkworkwork

Input 3:
ball

Output 3:
ballballballball
