3. Extract File
Write a program that reads the path to a file and subtracts the file name and its extension.

Examples:

Input 1:
C:\Internal\training-internal\Template.pptx

Output 1:
File name: Template
File extension: pptx

Input 2:
C:\Projects\Data-Structures\LinkedList.cs

Output 2:
File name: LinkedList
File extension: cs
