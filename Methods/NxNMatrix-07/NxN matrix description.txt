7. NxN Matrix
Write a method that receives a single integer n and prints nxn matrix with that number.

Examples:

Input 1:
3

Output 1:
3 3 3
3 3 3
3 3 3

Input 2:
7

Output 2:
7 7 7 7 7 7 7	
7 7 7 7 7 7 7
7 7 7 7 7 7 7
7 7 7 7 7 7 7
7 7 7 7 7 7 7
7 7 7 7 7 7 7
7 7 7 7 7 7 7

Input 3:
2

Output 3:
2 2
2 2
