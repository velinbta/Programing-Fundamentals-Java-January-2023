2. Grades
Write a method that receives a grade between 2.00 and 6.00 and prints the corresponding grade in words:

2.00 - 2.99 - "Fail"
3.00 - 3.49 - "Poor"
3.50 - 4.49 - "Good"
4.50 - 5.49 - "Very good"
5.50 - 6.00 - "Excellent"

Examples:

Input 1:
3.33

Output 1:
Poor

Input 2:
4.50

Output 2:
Very good

Input 3:
2.99

Output 3:
Fail
