9. Greater of Two Values
You are given two values of the same type as input. The values can be of type int, char of String. Create a method getMax() that returns the greater of the two values: 

Examples:

Input 1:
int
2
16

Output 1:
16

Input 2:
char
a
z

Output 2:
z

Input 3:
string
Ivan
Todor

Output 3:
Todor
