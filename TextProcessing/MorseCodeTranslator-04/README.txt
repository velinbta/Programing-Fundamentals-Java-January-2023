4. *Morse Code Translator
Write a program that translates messages from Morse code to English (capital letters). Search for site to help you (without the numbers from morse code). The words will be separated by a space (' '). There will be a "|" character which you should replace with ' ' (space).

Examples:

Input 1:
.. | -- .- -.. . | -.-- --- ..- | .-- .-. .. - . | .- | .-.. --- -. --. | -.-. --- -.. .

Output 1:
I MADE YOU WRITE A LONG CODE

Input 2:
.. | .... --- .--. . | -.-- --- ..- | .- .-. . | -. --- - | -- .- -..

Output 2:
I HOPE YOU ARE NOT MAD
